Setting up the Build System¶

To modify Firmware for a node controller you will need Zig installed. To manage Zig version we will install ZVM.

  1. Run curl https://www.zvm.app/install.sh | bash
  2. Run source /home/aether/.bashrc to start using ZVM in the same terminal
  3. Run zvm install 0.15.1 to install the specific version of zig we need.

Zig should now be installed. To test that firmware compilation is working switch to the firmware directory with cd NCv4_Firmware and run zig build (this will take several minutes). This command will recompile the Web Assembly Modules used for the Influence Engines and Logic Editor. To get the compilation system ready to build firmware for our Node Controller Devices, we also need to install PlatformIO.

  1. run the included python script to install Platform IO, python3 get-platformio.py
  2. add pio to the PATH. run sudo nano ~/.bashrc
  3. scroll to the bottom and add the line export PATH="$PATH:$HOME/.platformio/penv/bin" use CTRL X to exit, Y to confirm changes and ENTER to complete the addition
  4. run source ~/.bashrc for these changes to take effect in this terminal

Platform IO should now be installed, you can test its working by running zig build arduino for compile firmware for Node Controller devices

You are now setup to edit Aether's Influence Engines and Logic Reactor Blocks and compile them to Web Assembly for the user interface, and to device firmware.
