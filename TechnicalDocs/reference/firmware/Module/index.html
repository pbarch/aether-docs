Module¶

Description¶

The Module class is an interface that all modules must implement to be registered in the firmware or web client

API Refrence¶

Header File¶

#include "Module.h"

Member Functions¶

virtual void receiveEvent(const char* eventName, table tb, uint64_t millis);
Receive an module event with arbitrary parameters stored in tb. Any pointers should not be stored for use after this function, as both eventName and table parameters will be freed after an event is received
