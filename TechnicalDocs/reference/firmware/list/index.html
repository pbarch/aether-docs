list¶

Description¶

A growable list that acts similarly to std::vector but only depends on the c standard library not cpp

It can be iterated over using a for each loop, and a refrence to items can be accessed using the [] operator

API Refrence¶

Header File¶

#include "list.h"

Definition¶

template<typename T> struct list;

Member functions¶

void push(T item);
Allocates if space needed for item and adds item to the end of the list.

T* get(int index);
Returns a pointer to the item at index.

void remove(int index);
Removes the item at index from the list and moves items down to fill in the empty space.

int len();
Returns the amount of items in the list.

void free();
Deallocates the internal item storage, if T is dynamically allocated then each item must be freed, if the list struct is dynamically allocated then the caller is responsible for freeing that memory

Examples¶

list<int> numbers;
numbers.push(12);
numbers.push(15);
numbers.push(7)
numbers.remove(1);
// numbers: 12, 7

bool is_true = numbers[1] == 7;
bool is_also_true = *numbers.get(1) == 7;

for (int& number : numbers) {
  number = number*2;
}
// numbers: 24, 14
numbers[1] // 14
